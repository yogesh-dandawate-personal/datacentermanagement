================================================================================
TROUBLESHOOTING
================================================================================

Problem: 500 Internal Server Error
  → Check DATABASE_URL in Environment Variables
  → Make sure database is running and accessible

Problem: Deployment taking too long
  → First deployment is slow (installs packages)
  → Wait up to 5 minutes

Problem: Domain not working
  → Wait 30 seconds for DNS
  → Clear browser cache (Ctrl+Shift+Delete)
  → Try incognito window

Problem: "Module not found"
  → Vercel auto-installs from requirements.txt
  → Check dependencies are committed

LOGS:
  → Dashboard > Deployments > [Latest] > Logs tab
  → Check for error messages
